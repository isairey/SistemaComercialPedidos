# 🏢 Sistema Empresarial de Pedidos de Venta

Plataforma web diseñada para la gestión eficiente de pedidos de venta dentro de una empresa, permitiendo controlar clientes, productos, inventario y procesos comerciales en un solo sistema.

---

## 🚀 Descripción

El **Sistema Empresarial de Pedidos de Venta** es una solución orientada a negocios que buscan optimizar su flujo de ventas, automatizar procesos y mantener un control centralizado de sus operaciones comerciales.

Incluye funcionalidades para la administración de usuarios con roles, gestión de pedidos, seguimiento de ventas y generación de reportes.

---

## 🎯 Características principales

* 👤 **Gestión de usuarios y roles**

  * Administrador
  * Vendedor
  * Supervisor

* 📦 **Gestión de productos**

  * Registro, edición y eliminación
  * Control de precios y stock

* 🧾 **Pedidos de venta**

  * Creación y seguimiento de pedidos
  * Estados de pedido (pendiente, procesado, entregado)

* 📊 **Reportes y estadísticas**

  * Ventas por periodo
  * Rendimiento de vendedores
  * Historial de pedidos

* 🔐 **Autenticación y seguridad**

  * Inicio de sesión
  * Control de acceso por roles

* 📱 **Diseño responsivo**

  * Compatible con dispositivos móviles y escritorio

## ⚙️ Instalación

1. Clonar el repositorio:

```bash id="f3k2lz"
git clone https://github.com/isairey/SistemaComercialPedidos.git
```

2. Acceder al proyecto:

```bash id="g8p4mx"
cd SistemaComercialPedidos
```

3. Instalar dependencias:

```bash id="n2r7st"
npm install
# o
composer install
```

4. Configurar variables de entorno (`.env`)

5. Ejecutar el servidor:

```bash id="w5k9dq"
npm start
# o
php artisan serve
```

## 🧪 Uso del sistema

1. Iniciar sesión con credenciales válidas
2. Registrar productos y clientes
3. Crear pedidos de venta
4. Monitorear el estado de los pedidos
5. Generar reportes de ventas

Este sistema busca mejorar la organización y eficiencia en la gestión de ventas, reduciendo errores manuales y facilitando la toma de decisiones mediante datos en tiempo real.
